DOM Frequently Asked Question

1. What is DOM and it's purpose ?

DOM stand for Document object mode. The document object model is a programming interface that manipulate HTML documents. With DOM, the structure, style and content of the document that is the entire HTML can be change.

2. How to select Element in DOM ?

There are 5 ways in which you can select elements in a DOM using selectors.

  • document.getElementByTagName(): The method selects all the tags in the documents.
  • document.getElementById(): The method selects specific Id in the documents.
  • document.getElementByClassName(): The method selects all of the class which used in the documents.
  • querySelector(): The method selects Id and Class which is specified in the documents. If it's Id then we have to use "#" and it's class then use "."
  • querySelectorAll(): The method selects all of the elements in the documents.

    • 3. What is Event bubbling ?

    Function that are executed when a button is clicked are called events. Now when the button is clicked, in the first step events finds out where is clicked in the documents. The second step is target the element in the documents. The third step is find out where the event handler is added from the last element of the documents. When something is added to the water, some bubbles rise to the top and it's similar to the event bubbling method that's why it's called Event Bubbling.

    4. What is Callback function and it's purpose ?

    When a function is created with input parameters and it's value is returned, and when the function is called through the return value, it is called a callback function. By using callback function, one function does not have to be written more than once.
